¿Cuándo desea utilizar un repositorio remoto en lugar de mantener todo su trabajo local?

	cuando colaboro con mas gente o para poder acceder a el desde cualquier 	ubicacion o como respaldo

¿Por qué puede querer tirar siempre los cambios manualmente en lugar de tener Git mantenerse automáticamente actualizados con su repositorio remoto?

	para mayor control y evitar que algo pase a produccion cuando
	no deberia

Describir las diferencias entre las Forks, clones y ramas. ¿Cuándo utilizar uno en lugar de otro?

	forks cuando se quiere tener una copia de un repositorio en github de uno
	ubicado tambien en github.
	clone se utiliza para clonar un repositorio en mi equipo
	y las ramas para cuando se quiere trabajar sin afectar la rama principal

¿Cuál es el beneficio de tener una copia del último estado conocido de la distancia almacenada localmente?

	poder conocer el estado actual entre el repositorio local y el de github
	con el fin de sincronizarlos cuando sea necesario
